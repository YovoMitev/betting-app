# Simple Betting App Client - React Typescript Vite Tailwind

### Setup:

#### Environment

Create `.env` file in the root of the respository. For easy setup you can copy the .env:

```bash
VITE_API_HOST=http://localhost:3001
```

## Installation

#### Yarn

```bash
# Install dependencies
yarn
```

#### Npm

```bash
# Install dependencies
npm install
```

## Running the app

```bash
# development:npm
$ npm run dev

# development:yarn
$ yarn dev
```
